Welcome to VECTOR

VECTOR is a simple game engine, with a simple and extended api for graphics, animation, armature and physicis.

It use SDL2 and is image and font extension as graphics engine, adding to that an extra abstraction layer, with a simplier and extended interface.

Get the project 
$ git clone https://github.com/Mandarancio/Vector.git
$ cd Vector

I sudgest to use Eclipse with the CDT plugin to compile and use it (the project configuration is already included).1
